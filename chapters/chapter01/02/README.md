# How To Execute

## low_open.c

```bash
$ gcc low_open.c -o low_open
$ ./low_open
$ cat data.txt
```

## low_read.c
```bash
$ gcc low_read.c -o low_read
$ ./low_read
```

## fd_seri.c
```bash
$ gcc fd_seri.c -o fd_seri
$ ./fd_seri
```
